# 贪吃蛇游戏

一个基于 TypeScript 和 Canvas 开发的经典贪吃蛇游戏。

## 游戏特色

- 三种难度等级(简单、普通、困难)
- 皮肤商店系统
- 金币奖励机制
- 背景音乐和音效
- 流畅的动画效果
- 高分记录保存

## 操作说明

### 主菜单
- 1-3: 选择难度并开始游戏
- S: 进入商店
- O: 进入设置

### 游戏中
- 方向键/WASD: 控制蛇的移动
- P: 暂停/继续游戏
- R: 游戏结束时重新开始

### 商店
- 1-3: 购买皮肤
- U: 切换已拥有的皮肤
- R: 领取每日奖励
- B/ESC: 返回主菜单

### 设置
- S: 开启/关闭音效
- M: 开启/关闭音乐
- B/ESC: 返回主菜单

## 技术栈

- TypeScript
- HTML5 Canvas
- Web Audio API
- LocalStorage
